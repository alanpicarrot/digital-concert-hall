# 數位音樂廳 - 用戶前台 (Digital Concert Hall Client)

數位音樂廳平台的用戶前台應用程式，提供一般使用者瀏覽音樂會訊息、購票和觀看內容的介面。

## 功能特點

- 音樂會資訊瀏覽
- 會員註冊與登入
- 購票與訂單管理
- 直播與點播內容觀看

## 開發環境設置

1. 安裝依賴：
   ```
   npm install
   ```

2. 啟動開發服務器：
   ```
   npm start
   ```

應用將在 http://localhost:3000 上運行。

## 目錄結構

- `src/components` - 可重用的 UI 元件
- `src/pages` - 頁面元件
- `src/layouts` - 頁面布局元件
- `src/services` - API 服務和數據處理
- `src/router` - 路由設置
- `src/contexts` - React Context API 上下文

## 技術棧

- React 19
- React Router 7
- Axios
- Tailwind CSS
