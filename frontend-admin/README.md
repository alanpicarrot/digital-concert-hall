# 數位音樂廳 - 管理後台 (Digital Concert Hall Admin)

數位音樂廳平台的管理後台應用程式，提供管理員上架音樂會、管理票種、審核訂單和查看數據分析的介面。

## 功能特點

- 演出和音樂會管理
- 票種設置和管理
- 訂單審核與處理
- 系統數據分析
- 用戶管理

## 開發環境設置

1. 安裝依賴：
   ```
   npm install
   ```

2. 啟動開發服務器：
   ```
   npm start
   ```

應用將在 http://localhost:3001 上運行。

## 目錄結構

- `src/components` - 可重用的 UI 元件
- `src/pages` - 頁面元件
- `src/layouts` - 頁面布局元件
- `src/services` - API 服務和數據處理
- `src/router` - 路由設置
- `src/contexts` - React Context API 上下文

## 技術棧

- React 19
- React Router 7
- Axios
- Tailwind CSS
